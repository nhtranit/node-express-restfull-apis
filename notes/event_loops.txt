Javascript : Single Thread (Đơn luồng) LIFO(Last In First Out)

chạy trình tự
function demo(){
    alert(1);
    console.log(2);// only run after done exec line above
}

Event loop là cơ chế giúp Javascript có thể thực hiện nhiều thao tác cùng một lúc (concurrency)
